## ⚙️ Complexity


Best Case → O(n log n)
Average Case → O(n log n)
Worst Case → O(n log n)
Space → O(n)


---

## 🧠 Key Idea

Merge Sort guarantees predictable performance.

---

## 📌 Strength

- Stable sorting
- Consistent performance
- Ideal for large datasets

---

## ⚠️ Weakness

- Extra memory required
