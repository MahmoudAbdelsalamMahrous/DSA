## 🧠 Concept

Pick a pivot → Partition → Recursively sort

## ⚙️ Complexity


Best Case → O(n log n)
Average Case → O(n log n)
Worst Case → O(n²)
Space → O(log n)


---

## 🧠 Key Insight

Performance depends entirely on pivot selection.

Good pivot → fast  
Bad pivot → quadratic

---

## 📌 Strength

- Fast in practice
- In-place sorting
- Widely used in real systems

---

## ⚠️ Risk

Worst-case degradation if poorly implemented
